## Install

[Download](https://github.com/kwrap/kwrap-cli/releases) the binary from the release page

Or use `cargo` to install

```bash
cargo install kwrap
```

## Usage

```sh
# Start
kwrap

# Clear all info
kwrap logout

# Print config info
kwrap info
```
